# Portfolio – Emma Arends

Persönliche Portfolio-Website, gebaut mit reinem HTML, CSS und JavaScript – ohne Frameworks oder Build-Tools.

## Features

- **Responsives Design** für Desktop und Mobil
- **Sprachumschalter** (Deutsch / Englisch) für die komplette Seite
- **Scroll-Animationen** mit geschwungenen Wellen-Übergängen zwischen den Abschnitten
- **Tippeffekt** im Hero-Bereich
- **Skill-Übersicht** als Tag-Wolke, gruppiert nach Kategorien
- **Werdegang** als interaktive Timeline
- **Projekt-Karussell** zum Durchklicken
- **Kontaktformular** mit Validierung

## Aufbau

```
.
├── index.html      # Seitenstruktur und Inhalte
└── CSS/
    └── styles.css  # Styling und Animationen
```

## Lokal öffnen

Repository klonen und `index.html` im Browser öffnen:

```bash
git clone https://github.com/emarends/<repo-name>.git
cd <repo-name>
```

Dann `index.html` per Doppelklick öffnen – es wird kein Server benötigt.

## Verwendete Technologien

HTML5 · CSS3 · JavaScript (Vanilla)
